Research / Academic Work

The Telepathy Project - This is a parapsychology research project I've been working on with Dr Diane Powell. She's investigating claims of psychic abilities in autistic children. I've contributed to this research by writing GUI applications in Python to display random numbers, letters, and words, for use in the telepathy tests.

Economic History Research - When I was in college, I worked as a research assistant for Professor Thomas Sargent. He was researching economic issues related to the financing of the American Revolution, specifically the refinancing of debt the US had taken on during the war. I contributed to this work by writing programs in Python to clean and analyze financial data from the 1700s. I also spent a lot of time translating Matlab code into Python.

Translating Class Notes - During my last year of college, I helped one of the professors in the Economics department translate all his class notes from Matlab to Python. This was the final product. This was a for a financial analysis class, so this project gave me some experience with economic modeling, using techniques like Auto-regression moving average models, Markov chains, and stuff like that.
